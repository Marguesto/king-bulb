Story

Once upon a time, King Unnamed of Nowhereland was both smiling and crying at the same time. He was smiling because his enormous brand-new palace with many spacious rooms and corridors has just been finished. He is also crying as these rooms need to be illuminated and kept warm, but the ongoing increase of utility costs affect him as well. So now it's time to think about the placement of the palace's light bulbs. We need to place them so that everything is properly lit, but we cannot install any unnecessary bulbs.

Instructions

  • Timer starts when you add user.
  • Click a level button to start the game, or restart it.
  • Click a cell to place/remove a bulb. bulbs can only be placed above white tiles.
  • Light doesn't spread diagonaly, only straight.
  • Number on black cells indicates the required number of adjacent bulbs.
  • Two light bulbs can NEVER illuminate each other..
  • Illuminate the white tiles.

Score board

Creating levels?

It's possible to create levels by writing a string to express the level as the next placeholder expresses the displayed level. # for white tiles. 0, 1, 2, 3 for black tiles with numbers, and @ for non-numbered black tiles. the width must equal the height.
